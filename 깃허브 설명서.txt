깃허브와 연동된 폴더의 변경사항 체크 명령어
git status

수정된 여러 파일을 한꺼번에 수정하는법 ( 로컬에서의 변경사항을 깃허브로 업로드 하는법 ) 
git add .
git commit -m "message"  //코멘트 다는부분
git push origin master // 이거 입력하면 로그인을 하라고 뜬다 YangJiUng 입력하면 비번 입력하면 ㅇㅋ

pull 하는법 ( 깃허브에서 변경된 사항을 로컬폴더에 옮기는 작업 )
git pull origin master
